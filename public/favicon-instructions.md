# Grid Logic Favicon Setup

## Quick Setup Instructions

### Option 1: Use Favicon.io (Recommended)
1. Go to https://favicon.io/favicon-generator/
2. **Text Settings**:
   - Text: "GL" or "9"
   - Background: Rounded Rectangle
   - Font Family: Noto Sans (Bold)
   - Font Size: 50-60
   - Background Color: #6366f1 (indigo)
   - Text Color: #ffffff (white)

3. Download the generated package
4. Replace the files in the `public` folder
5. Update the HTML links (see below)

### Option 2: Grid Pattern Design
Create a 3x3 grid pattern:
- Canvas: 32x32px
- 9 squares (each ~8px)
- Fill pattern: checkered or strategic placement
- Colors: Indigo (#6366f1) and white

### Option 3: Professional Service
- Fiverr: $5-15 for custom favicon
- 99designs: Professional contest
- Upwork: Hire a designer

## HTML Implementation

Replace the current favicon link in `index.html`:

```html
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="manifest" href="/site.webmanifest">
```

## File Requirements

You'll need these files in your `public` folder:
- favicon.ico (16x16, 32x32, 48x48)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)
- android-chrome-192x192.png
- android-chrome-512x512.png
- site.webmanifest

## Brand Guidelines

**Colors**: 
- Primary: #6366f1 (indigo-500)
- Secondary: #8b5cf6 (purple-500)
- Text: #ffffff (white)

**Style**: 
- Modern, geometric
- Clean lines
- Professional but approachable
- Consistent with app branding
